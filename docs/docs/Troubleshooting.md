Plugin Logs
----------------
Download the Plugin logs using the following methods:  

- Using dvp  
dvp download-logs -c plugin_config.yml -e <Delphix_Engine_Name> -u admin --password <password>  

- Using GUI  
Help --> Supports Logs --> Plugin Logs --> Download
